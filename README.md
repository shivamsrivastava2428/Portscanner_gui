# Lodha Port Scanner GUI 🔍

A simple yet powerful **GUI-based port scanner** built with Python and Nmap.  
Designed to help security analysts and students quickly scan hosts and visualize results in an intuitive interface.

---

## 🚀 Features
- Scan single or multiple IP addresses
- Choose custom port ranges
- Fast and efficient scanning using Nmap
- User-friendly GUI for easy interaction
- Export results to text/CSV for reporting

---

## 🛠️ Tech Stack
- **Python 3**
- **Tkinter** (GUI)
- **Nmap** (backend scanning engine)
